# Portfólio Samuel Teixeira

Site desenvolvido para apresentar o trabalho de Samuel Teixeira como editor de vídeo para YouTube, conteúdo gamer, entretenimento e storytelling.

## Sobre o projeto

Este projeto é uma landing page responsiva criada com HTML, CSS e JavaScript puro.  
O objetivo principal é apresentar os serviços de edição de vídeo, exibir trabalhos selecionados e direcionar possíveis clientes para contato via WhatsApp.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- YouTube Embed

## Funcionalidades

- Layout responsivo para desktop, tablet e smartphone
- Seção de showreel com vídeo incorporado do YouTube
- Cards de trabalhos com thumbnails automáticas do YouTube
- Menu mobile
- Animações de entrada
- Botão flutuante de WhatsApp
- Links para redes sociais
- Favicon personalizado
